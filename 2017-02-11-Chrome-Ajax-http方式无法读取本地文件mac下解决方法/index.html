Chrome+Ajax:$http方式无法读取本地文件mac下解决方法

  • 2017-02-11
  • ·
  • JavaScript
  • ·
  • Views

环境

OSX、Chrome

解决方法

  • 配置chrome的启动参数

    • 命令行实现：chrome可执行文件所在位置/可执行文件 + 配置启动参数

      /Applications/Google\ Chrome.app/Contents/MacOS是chrome所在路径

      Google\ Chrome是可执行文件

      –allow-file-access-from-files是启动参数

    • 编写shell脚本(实际上和命令行实现无区别)

      “$@”将参数独立分开，独立打印
