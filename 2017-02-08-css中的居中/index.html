原文：Centering in CSS: A Complete Guide

水平居中

  • 内联元素：通过text-align属性设置其基于父元素对齐

  • 块级元素：当设置了宽度时，可以将margin-left和margin-right设置为auto，使其水平对齐。但宽度设置为100%时，将看不到对齐效果

  • 不止一个块级元素：将块级元素的display属性设置为inline-block，利用内联的性质将其设为居中

    各个块级框并排排列

垂直居中

  • 内联元素

    • 如果是单行：可以通过设置padding-top、padding-bottom为相同的值，实现垂直居中；因为某种原因padding没有办法用的时候，将line-height设置为height也可以实现居中效果
    • 多行：可以设置padding-top和padding-bottom值相等；如果不行，将display设置为table-cell，vertical-align属性设置为middle实现垂直居中

  • 块级元素

    • 如果知道元素高度：可以根据具体的值进行设置（不能适用多种设备，太过绝对化）
    • 不知道元素高度：设置其高度为父元素高度的一半，再将其上移自身高度的一半（利用transform性质）

同时设置水平、垂直居中

  • 如果知道具体的宽和高：通过设置具体的值使其实现居中效果
  • 不知道具体宽和高：通过transform性质进行设置
